Introduction

Welcome, my name is Paras Bhattrai. I am a software developer from Ashoka University who loves to build technical solutions for businesses to help them realize their goals efficiently. I appreciate good design, simple and easy to use interfaces.

I usually adapt to tech stack that solves the problem at hand most effectively but I usually build web interfaces in either Java stack or in Node.js with database in PostgreSQl or Mongo.
